sdpctl configure signin

Sign in and authenticate to Collective

Synopsis

Sign in to the Collective using the configuration file created by the 'sdpctl configure' command. This will fetch a token on valid authentication which will be valid for 24 hours and stored in the configuration.

sdpctl configure signin [flags]

Examples

  # default sign in command
  > sdpctl configure signin
